# Student Forex & Crypto Advisor

A simple website that gives **educational financial advice** for students who want a safer structure before buying forex or crypto.

## What it does

- Ranks forex/crypto instruments by a transparent score.
- Recommends a starter instrument based on risk tolerance and goal.
- Suggests position sizing, max risk per trade, and stop-loss guidance.

## Run locally

```bash
python3 -m http.server 8000
```

Open: `http://localhost:8000/index.html`

## Reminder

This tool is for education only and is **not licensed personal financial advice**.
